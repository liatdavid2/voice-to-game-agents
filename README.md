# AI-Powered Game Generation Platform

AI-Powered Game Generation Platform is an agent-based system that creates personalized browser games for children from natural language or voice input.

The platform receives a game idea, generates a complete playable web game, saves it as a versioned project, and optionally publishes it to GitHub Pages.

The system is designed as an agentic workflow, where each agent is responsible for a specific stage in the game creation lifecycle.

## Demo Video

This demo shows how a user can describe a game idea, generate a playable browser game, save it as a new version, and open it from the game library.
